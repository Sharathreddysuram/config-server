# Banking Microservices System

Enterprise-level Banking Application built using Spring Boot Microservices Architecture.

## Technologies Used
- Java 17
- Spring Boot
- Spring Cloud
- Eureka Server
- API Gateway
- OpenFeign
- Spring Security
- JWT Authentication
- MySQL
- Maven

## Microservices
- Auth Service
- Customer Service
- Account Service
- Transaction Service
- API Gateway
- Service Registry

## Features
- Customer Management
- Account Management
- Deposit & Withdraw Transactions
- JWT Authentication
- Service Discovery
- Inter-service Communication
- Centralized API Gateway
- Exception Handling

## Architecture
Client → API Gateway → Microservices → MySQL
